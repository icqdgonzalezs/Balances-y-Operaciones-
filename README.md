# 🇪🇸 ESPAÑOL

## Descripción General

Este repositorio contiene una colección de modelos de ingeniería química y de procesos desarrollados en Python, enfocados en balances de masa, balances de energía, operaciones unitarias y análisis de datos industriales.

El objetivo es utilizar Python como una herramienta de ingeniería para el análisis de procesos, simulación y apoyo a la toma de decisiones operacionales, similar a herramientas utilizadas en la industria como Excel, Aspen o MATLAB.

---

## Problema de Ingeniería

Los procesos industriales requieren análisis continuo de:

* Balances de masa
* Consumo energético
* Desempeño de equipos
* Eficiencia de procesos
* Datos operacionales
* Desempeño productivo

Muchos de estos análisis se realizan manualmente o en planillas Excel. Este repositorio muestra cómo Python puede utilizarse para modelar, simular y analizar procesos industriales de forma más eficiente y reproducible.

---

## Metodología

Los modelos desarrollados en este repositorio siguen una metodología típica de ingeniería:

1. Definición del proceso
2. Supuestos y condiciones de operación
3. Formulación de balances de masa y/o energía
4. Implementación del modelo matemático
5. Simulación y análisis paramétrico
6. Interpretación de resultados ingenieriles

El enfoque no es solo la programación, sino el análisis de ingeniería y la interpretación de resultados.

# 🇺🇸 ENGLISH

## Project Overview

This repository contains a collection of chemical and process engineering models developed in Python, focused on mass balances, energy balances, unit operations, and industrial data analysis.

The objective is to use Python as an engineering tool for process analysis, simulation, and operational decision support, similar to tools used in industry such as Excel, Aspen or MATLAB.

---

## Engineering Problem

Industrial processes require continuous analysis of:

* Mass balances
* Energy consumption
* Equipment performance
* Process efficiency
* Operational data
* Production performance

Many of these analyses are performed manually or in spreadsheets. This repository shows how Python can be used to model, simulate and analyze industrial processes in a more efficient and reproducible way.

---

## Methodology

The models developed in this repository follow a standard engineering approach:

1. Process definition
2. Assumptions and operating conditions
3. Mass and/or energy balance formulation
4. Mathematical model implementation
5. Simulation and parametric analysis
6. Interpretation of engineering results

The focus is not only programming, but engineering analysis and interpretation of results.
